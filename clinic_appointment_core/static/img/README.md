# Social Sharing Images (TASK-F3-005)

This directory contains images used for social media sharing (Open Graph).

## Required Images

### default_appointment_share.png
- **Purpose**: Default image for appointment booking pages when shared on social media
- **Recommended size**: 1200x630 pixels (Facebook/LinkedIn recommended size)
- **Format**: PNG or JPG
- **Content suggestions**:
  - Clinic logo
  - Text: "Book Your Appointment Online"
  - Professional, clean design
  - High contrast for readability

## Creating the Image

You can create this image using:
- Canva (canva.com) - Easy online design tool
- Adobe Photoshop/Illustrator
- GIMP (free alternative)

## Example Template

```
+----------------------------------+
|   [Clinic Logo]                  |
|                                  |
|   Book Your Appointment          |
|   Online                         |
|                                  |
|   Fast • Easy • Convenient       |
+----------------------------------+
     1200 x 630 pixels
```

## Note

Until this image is created, the Open Graph tags will reference this path but may not display an image on social media platforms. This won't break functionality, but social media posts will lack a preview image.
